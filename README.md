React Server-Side Rendering

Component Design - Carousel React component

### To Install & Run

npm install

npm run build:watch

npm run build:watch:client

nodemon ./build/server/index.js


### run in dev

npm run build:watch 

npm run build:watch:client 

npm run start:dev 

### prod

npm run build 

npm run build:client 

npm run build:prod 

npm run start
